# bookstoreAPI

### Authors

* get /authors
* get /authors/:id 
* post /authors 
* put /authors/:id 
* delete /authors/:id 

### Books

* get /books 
* get /books/search 
* get /books/:id 
* post /books 
* put /books/:id 
* delete /books/:id 

#### This API uses Node.js, Express, MongoDB
#### Must creat an .env file with DB_CONNECTION_STRING
